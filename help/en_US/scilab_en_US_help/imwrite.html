imwrite

BWrites an image from Scilab to Disk

Calling Sequence

imwrite ( input_image , string path);

Arguments

input_image:

the image matrix which has to be saved

path:

The path, including the name of the file in which the image has to be written. Note that in the example below, the file in which the image is written is "newimg.jpg"

Description

The function writes an image to a file

Example

a = imread("/home/Pictures/img.jpg");
imwrite(a,"/home/Desktop/newimg.jpg");
