gaussianblur

Blurs an image using a Gaussian filter

Calling Sequence

output_image = gaussianblur(input_image,int ksize_height,int ksize_width,double sigmaX,double sigmaY);

Arguments

input_image:

the image matrix on which blur has to be applied

ksize_height:

height of gaussian kernel size

ksize_width:

width of gaussian kernel size

sigmaX:

Gaussian kernel standard deviation in x-direction

sigmaY:

Gaussian kernel standard deviation in y-direction

Description

The function convolves the source image with the specified Gaussian kernel. In-place filtering is supported

Example

a = imread("/home/Pictures/img.jpg");
b=gaussianblur(a,3,3,0,0);
imshow(b);
