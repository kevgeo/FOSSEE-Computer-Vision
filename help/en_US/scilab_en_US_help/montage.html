montage

Creates a montage with a list of images

Calling Sequence

output_image = montage(List input_images, int no_of_rows, int no_of_cols);

Arguments

input_images:

A list of images which have to be included in the montage

no_of_rows:

number of rows in montage

no_of_cols:

number of columns in montage

Description

The function creates a montage with the list of images. Montage is created row-wise with images taken in order from the list. Note that the following condition should hold true:- no_of_rows*no_of_cols >= no_of_images > (no_of_rows-1)*(no_of_cols)

Example

a = imread("/home/Pictures/img.jpg");
b = imread("/home/Pictures/img2.jpg");
l = list(a,b,b,a);
c = montage(l,2,2);
imshow(c);
