getTextSize

Calculates the width and height of a text string

Calling Sequence

[size,baseline] = getTextSize(string input_text,string fontface,double fontscale,int thickness);

Arguments

input_text:

Input text string

fontface:

Font type. One of:-

  • FONT_HERSHEY_SIMPLEX
  • FONT_HERSHEY_PLAIN
  • FONT_HERSHEY_DUPLEX
  • FONT_HERSHEY_COMPLEX
  • FONT_HERSHEY_TRIPLEX
  • FONT_HERSHEY_COMPLEX_SMALL
  • FONT_HERSHEY_SCRIPT_SIMPLEX
  • FONT_HERSHEY_SCRIPT_COMPLEX
where each of the font IDâ€™s can be combined with FONT_ITALIC to get the slanted letters.

fontscale

Font scale factor that is multiplied by the font-specific base size

thickness

Thickness of the lines used to draw a text

Description

The function getTextSize calculates and returns the size of a box that contains the specified text

Example

[a,b] = getTextSize("Image","FONT_HERSHEY_SIMPLEX",5,8);
