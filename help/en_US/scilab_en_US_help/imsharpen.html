imsharpen

Sharpens the Image

Calling Sequence

output_image = imsharpen(input_image);

Arguments

input_image:

the image that has to be sharpened

Description

The function uses the unsharp masking method to sharpen the image. A gaussian smoothing filter is used and then subtracted from the smoothed version from the original image (in a weighted way so the values of a constant area remain constant).

Example

b = imsharpen(a);
imshow(b);
