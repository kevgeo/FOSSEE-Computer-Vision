Image Processing Toolbox

  • blur — Blurs an image using the normalized box filter.
  • canny — Finds edges in an image using the Canny86 algorithm.
  • cornerEigenValsAndVecs — Calculates eigenvalues and eigenvectors of image blocks for corner detection.
  • CornerHarris — Harris edge detector.
  • filter2D — Convolves an image with the kernel.
  • gaussianblur — Blurs an image using a Gaussian filter
  • getTextSize — Calculates the width and height of a text string
  • HoughLines — Finds lines in a binary image using the standard Hough transform..
  • HoughLinesP — Finds line segments in a binary image using the probabilistic Hough transform.
  • im2double — Converts pixel values from integer precision to floating-point precision
  • imcontrast — Changes contrast of input image
  • imcrop — Crops an image
  • imread — Loads an Image from a file, and returns the Image Matrix as a List
  • imresize — Changes the dimensions of an image
  • imsharpen — Sharpens the Image
  • imwrite — BWrites an image from Scilab to Disk
  • montage — Creates a montage with a list of images
  • Undistort — Transforms an image to compensate for lens distortion.
