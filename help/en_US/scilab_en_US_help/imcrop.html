imcrop

Crops an image

Calling Sequence

output_image = imcrop(input_image, int xvalue, int yvalue, int width, int height);

Arguments

input_image:

the image matrix on which crop function has to be applied

xvalue:

x-coordinate of top-left corner pixel

yvalue:

y-coordinate of top-left corner pixel

width:

width from the specified point

height:

height from the specified point

Description

The function crops the image using the specified values

Example

a = imread("/home/Pictures/img.jpg");
b=imcrop(a,10,10,100,100);
imshow(b);
