HoughLinesP

Finds line segments in a binary image using the probabilistic Hough transform.

Calling Sequence

List = houghlines(InputArray image, double rho, double theta, int threshold, double srn=0, double stn=0 )

Arguments

image:

8-bit, single-channel binary source image. The image may be modified by the function.

rho:

N Distance resolution of the accumulator in pixels.

theta:

Angle resolution of the accumulator in radians.

threshold:

Accumulator threshold parameter. Only those lines are returned that get enough votes ( >\texttt{threshold} ).

minLineLength:

Minimum line length. Line segments shorter than that are rejected.

maxLineGap:

Maximum allowed gap between points on the same line to link them.

Description

The function implements the probabilistic Hough transform algorithm for line detection.

Examples

image = imread("/media/Data-2/Pictures/pic.jpg");
a=houghlinesp( dst, 1, CV_PI/180, 80, 30, 10 );
