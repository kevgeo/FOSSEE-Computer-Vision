imcontrast

Changes contrast of input image

Calling Sequence

output_image = imcontrast(input_image,double alpha, double beta);

Arguments

input_image:

the image matrix on which blur has to be applied

alpha:

value of gain

beta:

value of bias

Description

The function changes contrast of an image using the alpha-beta filtering method: new_image = alpha * original_image + beta

Example

a = imread("/home/Pictures/img.jpg");
b=imcontrast(a,2.0,5.0);
imshow(b);
