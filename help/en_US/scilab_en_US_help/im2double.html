im2double

Converts pixel values from integer precision to floating-point precision

Calling Sequence

output_image = im2double(input_image);

Arguments

input_image:

the image matrix on which blur has to be applied

Description

The function converts pixel values from integer precision to floating-point precision(CV_64F)

Example

a = imread("/home/Pictures/img.jpg");
b=im2double(a);
