imresize

Changes the dimensions of an image

Calling Sequence

output_image = resize(input_image, int new_width, int new_height);

Arguments

input_image:

the image on which dimension changes have to be applied

new_height:

new height of image

new_width:

new width of image

Description

The function changes the dimensions of an images(reszies it) as per the user's requirement. Note that the aspect ratio may change if the user does not resize it in proportion

Example

a = imread("/home/Pictures/img.jpg");
b=imresize(150,200);
imshow(b);
